Este programa es una versióon actualizada y mejorada del juego base que se descargo en la libreria freegames para python

El programa base corre un juego llamado cannon, donde el jugador debera lanzar proyectiles que siguen una trayectoria parabólica.
Esto con el fin de golpear una serie de objetivos que salen de la parte derecha de la pantalla y avanzan progresivamente a la parte izquierda de la pantalla.
En el dado caso de que un objetivo no sea golpeado y logre llegar al lado izquierdo de la pantalla, el juego se detendra y el jugador habrá perdido.

Nuestros cambios hacen que
-Los objetivos vayan más rápido
-Los proyectiles vayan más rápido
-El juego nunca termine, los objetivos únicamente se reposicionan

El proyecto para la Semana Tec actualmente esta terminado y no requiere de más actualizaciones

En caso de que se quiera probar este juego se requiere de la instalación previa de Python y la libreria de FreeGames. 
Posteriormente se debe correr en la terminal de la computadora
